### Core Intent
Contextual variations are NOT lexical rephrasings (synonyms). They are lore-grounded identifiers, aliases, and narrative roles that a knowledgeable fan would use to describe the same truth.

### Instructions (Surgical Precision Rules)

1. **The Anchor-First Rule:** Every entry in `entity_refs` MUST be a direct substitute for the answer itself. 
   - DO NOT include victims, bystanders, or contextual characters from the question. 
   - *Example:* If the answer is "Bellatrix Lestrange," do NOT include "Sirius Black" or "Harry Potter" as variants.

2. **The "Non-Generic" Lore Rule:** Lore concepts must be DISCRIMINATIVE. 
   - Avoid generic categories like "Wizarding school," "Magical plants," or "Spells."
   - Include only concepts that explain the specific truth or mechanics of the answer (e.g., "Wand Allegiance," "Secret-Keeper," "Blood Pact").

3. **The Alias Cap:** - Max 2 aliases per entity. 
   - Prioritize the most common canonical alias (e.g., "Voldemort" ↔ "Tom Riddle").

4. **MCQ Strictness:** - Variations for MCQ must uniquely identify the correct option. 
   - If an alias overlaps with a distractor (e.g., using "Marauder" when both Sirius and Lupin are options), discard it.
