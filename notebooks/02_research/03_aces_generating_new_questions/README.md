# Content Factory: Question Generation – Research & Prototyping

This directory contains exploratory notebooks used to design and validate
the Phase 2 Content Factory question generation pipeline.

Notebooks here are considered prototypes and are not part of the production pipeline.

## Architectural impact

This research informed architectural decisions related to:
- "Shift Left" Enrichment
- Generative Model Selection (Gemini)
- Medallion Data Architecture
- Prompt standardization for batch pipelines

See the ADR index in the [Design Doc](../../../docs/00_DESIGN_DOC_AND_ARCHITECTURE.md) for the current decision records.
