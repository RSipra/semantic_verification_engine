# ADR-P2-00X: Exclusion of Yes/No (YN) Questions from Phase 2 Gold Dataset

**Status:** Accepted  
**Phase:** Phase 2 (Semantic Verification Tracer)

## Decision
Exclude Yes/No (YN) question types from the Phase 2 Gold dataset and runtime demo.

## Options considered
1. Include YN questions and apply semantic validation  
2. Retain YN questions with exact / fuzzy matching only  
3. Exclude YN questions from Phase 2 Gold dataset  

## Selected option
Option 3 (Exclude YN questions from Phase 2 Gold dataset)

## Rationale
Inspection of the legacy dataset showed that YN questions typically resolve to binary confirmations (e.g., “Yes, it can.” / “No.” / “True.”) with minimal explanatory content. These questions do not meaningfully benefit from semantic validation and do not stress the semantic verification logic under study in Phase 2. Including them would introduce noise without contributing to the primary learning objective.

## Trade-offs
**Pros:**
- Sharpens focus on question types where semantic equivalence matters (FR, EX)  
- Simplifies validation logic and demo narrative  
- Reduces noise in Gold dataset  

**Cons:**
- Reduced question-type coverage in Phase 2  
- YN questions are not demonstrated in the runtime demo  

## Risks
- Future readers may question reduced dataset diversity if the rationale is overlooked  

## Consequences & Future considerations
YN questions may be revisited in later phases if reformulated to require explanatory or evidence-based answers. This decision is scoped strictly to Phase 2 and does not preclude future support.
